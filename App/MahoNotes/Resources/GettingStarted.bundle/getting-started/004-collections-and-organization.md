# 4. Collections & Organization

Collections help you group related notes together. Think of them as folders with a bit more structure.

## Creating a Collection

Tap the **+** button in the sidebar and choose **New Collection**. Give it a name and an optional icon — that's it!

## Organizing Notes

Each note belongs to a collection. When you create a new note, choose which collection it goes into.

Within a collection, you can use **subdirectories** to further organize:

```
research/
  papers/
    quantum-gravity.md
    dark-matter.md
  ideas/
    new-experiment.md
japanese/
  grammar/
    particles.md
    verb-forms.md
  vocabulary/
    daily-words.md
```

## Tips

- **One vault per context** — e.g., one for personal notes, one for work
- **Collections for broad topics** — e.g., "Research", "Japanese", "Recipes"
- **Subdirectories for depth** — e.g., "Research / Papers / 2026"
