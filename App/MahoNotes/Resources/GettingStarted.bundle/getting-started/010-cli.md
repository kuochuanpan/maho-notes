# 10. Command-Line Interface (macOS)

Maho Notes includes a command-line tool called `mn` for power users on macOS. If you prefer working in the terminal, or want to automate your note workflow, this is for you.

## Installation

Install via [Homebrew](https://brew.sh):

```bash
brew tap mahopan/tap
brew install maho-notes
```

## Quick Start

```bash
mn init                     # set up your first vault
mn new "My Note"            # create a new note
mn list                     # list all notes
mn show my-note.md          # view a note
mn open my-note.md          # open in your editor ($EDITOR)
```

## Managing Notes

```bash
mn new "Title" --collection research    # create in a specific collection
mn meta my-note.md                      # view frontmatter
mn meta my-note.md --add-tag physics    # add a tag
mn delete old-note.md                   # move to trash
```

## Search

The CLI uses the same powerful CJK-aware search engine as the app:

```bash
mn search "quantum gravity"              # full-text search
mn search "漢字"                          # CJK search works too
mn search --collection japanese "助詞"    # search within a collection
```

## Vault Management

```bash
mn vault list               # list all vaults
mn vault add ~/my-vault     # add an existing vault
mn vault info               # show current vault details
mn collections              # list collections with note counts
mn stats                    # vault statistics
```

## Sync

```bash
mn config auth              # set up GitHub authentication
mn sync                     # pull + push changes
```

## Search Index

```bash
mn index                    # incremental rebuild
mn index --full             # full rebuild from scratch
```

## Tips

- The CLI and app share the same vault and data — changes in one are immediately visible in the other
- Use `mn` in scripts to batch-create or organize notes
- Pair with your favorite text editor for a keyboard-driven workflow
- Run `mn help <command>` for detailed options on any command
