# 9. 中文筆記範例

這是一篇中文範例筆記，展示 Maho Notes 對中文的完整支援。試試搜尋這篇筆記中的任何文字！

## 為什麼選擇 Markdown？

Markdown 是一種輕量級的標記語言，有幾個重要的優點：

1. **純文字格式** — 不依賴任何特定軟體，永遠可讀
2. **簡單易學** — 幾分鐘就能上手
3. **版本控制友好** — 可以用 Git 追蹤每一次修改
4. **跨平台** — 在任何裝置上都能編輯

## 數學公式

Maho Notes 支援 $\LaTeX$ 數學公式。例如，薛丁格方程式：

$$
i\hbar\frac{\partial}{\partial t}\Psi(\mathbf{r}, t) = \hat{H}\Psi(\mathbf{r}, t)
$$

或是更簡單的，質能等價公式：$E = mc^2$

## 學習筆記示範

> [!tip]
> 搜尋中文時，Maho Notes 能正確處理中文斷詞。試試搜尋「薛丁格」或「版本控制」！

## 小結

Maho Notes 從底層設計就支援中日韓文字的搜尋和顯示，不是事後補上的功能。這對多語言使用者來說非常重要。
