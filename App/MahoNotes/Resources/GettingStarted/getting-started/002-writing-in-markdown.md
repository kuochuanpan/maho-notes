# 2. Writing in Markdown

Markdown is a simple way to format text. If you've ever used `**bold**` or `# headings` in a chat app, you already know some markdown!

## Text Formatting

| What you type | What you get |
|---|---|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |

**Inline code** — wrap text in single backticks:

```markdown
Use `code` for inline code.
```

Result: Use `code` for inline code.

## Headings

Use `#` symbols to create headings:

```markdown
# Heading 1
## Heading 2
### Heading 3
```

## Lists

**Bullet lists:**

```markdown
- First item
- Second item
  - Nested item
```

**Numbered lists:**

```markdown
1. First step
2. Second step
3. Third step
```

**Task lists:**

```markdown
- [x] Completed task
- [ ] Pending task
```

## Links and References

```markdown
[Link text](https://example.com)
```

## Blockquotes

```markdown
> This is a quote.
> It can span multiple lines.
```

> This is a quote.
> It can span multiple lines.

## Horizontal Rules

Use three dashes to create a divider:

```markdown
---
```
