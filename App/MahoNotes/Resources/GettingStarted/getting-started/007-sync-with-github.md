# 7. Sync with GitHub

By default, your notes sync across your Apple devices via **iCloud** — no setup needed.

For additional flexibility, Maho Notes also supports **GitHub sync**.

## Why GitHub?

- **Cross-platform access** — View and edit your notes from any device with a browser
- **Version history** — Full git history for every change
- **Collaboration** — Share a vault with others via a GitHub repository
- **Backup** — An extra copy of your notes, independent of iCloud

## Setting Up GitHub Sync

1. Open **Settings** in Maho Notes
2. Sign in with your **GitHub account** (uses secure Device Flow — no password needed)
3. Choose an existing repository or create a new one
4. Tap **Sync** — done!

## How It Works

GitHub sync is **additive** to iCloud. Your notes still live on your device and sync via iCloud. GitHub provides a second sync channel for backup, sharing, or cross-platform access.

Sync is manual by default — tap the sync button when you want to push or pull changes. This keeps you in control.

## Not a Developer?

That's fine! GitHub sync is entirely optional. iCloud sync works out of the box with zero configuration. You can always enable GitHub sync later if you need it.
