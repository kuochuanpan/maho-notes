# Task Lists & Project Tracking

Markdown checkboxes turn any note into a lightweight task manager — no extra app needed.

## Basic Task List

```markdown
- [x] Define project scope
- [x] Set up repository
- [ ] Write first draft
- [ ] Review with team
- [ ] Ship it 🚀
```

- [x] Define project scope
- [x] Set up repository
- [ ] Write first draft
- [ ] Review with team
- [ ] Ship it 🚀

## Project Tracking Note

Here's a pattern for tracking a small project in a single note:

---

### 🎯 Goal

Build a personal website with a blog section.

### 📅 Timeline

| Milestone       | Target     | Status |
|-----------------|------------|--------|
| Design mockup   | March 18   | ✅ Done |
| Homepage        | March 22   | 🔧 In progress |
| Blog engine     | March 28   | ⏳ Planned |
| Launch          | April 1    | ⏳ Planned |

### ✅ This Week

- [x] Choose color palette
- [x] Set up Hugo
- [ ] Write "About" page
- [ ] Design navigation bar

### 📝 Notes

> [!tip]
> Keep one note per project. Use headings to separate phases.
> When the project is done, move the note to an "archive" collection.

---

## Why This Works

Maho Notes saves as plain markdown files. You can:

- **Search** across all your project notes instantly
- **Sync** via iCloud or GitHub — accessible on all your devices
- **Version control** — since notes are files, `git log` shows your project's history
